# מבחן מועד ב׳ – Movie Watchlist

Full stack exam project skeleton for a movie watchlist application.

## Technologies

- React with Vite
- JavaScript
- React Router
- Axios
- Tailwind CSS
- shadcn/ui basic setup
- Node.js
- Express
- Mongoose
- dotenv
- cors
- nodemon

## Run Frontend

```bash
cd client
npm install
npm run dev
```

## Run Backend

```bash
cd server
npm install
npm run dev
```

The backend uses `PORT` from `.env` or falls back to `5000`.
